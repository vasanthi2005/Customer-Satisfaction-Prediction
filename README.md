# Customer Satisfaction Prediction

## 📌 Project Overview
This project aims to predict customer satisfaction using machine learning techniques. The model analyzes customer support ticket data to identify patterns and predict satisfaction ratings.

---

## 📊 Dataset Description
The dataset contains customer support ticket information, including:

- Customer details (Age, Gender)
- Product purchased
- Ticket type and subject
- Ticket priority and channel
- Response time and resolution time
- Customer satisfaction rating (target variable)

---

## 🛠️ Steps Performed

### 1. Data Preprocessing
- Removed missing values in target column
- Filled missing data using forward fill
- Converted date column to datetime
- Encoded categorical variables

### 2. Feature Engineering
- Selected relevant features
- Removed unnecessary columns like date

### 3. Model Building
- Used Random Forest Classifier
- Split dataset into training and testing sets

### 4. Model Evaluation
- Accuracy Score
- Classification Report
- Confusion Matrix

### 5. Data Visualization
- Customer satisfaction distribution
- Ticket type distribution
- Ticket channel analysis
- Customer age distribution
